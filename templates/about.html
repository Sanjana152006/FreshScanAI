About

FreshScan AI

An intelligent food freshness detection system powered by Convolutional Neural Networks, built as a final-year engineering project using Django and TensorFlow.

Django 4.2 TensorFlow 2.13 MobileNetV2 CNN OpenCV

⚙️ How It Works

📷
1. Upload

User uploads a food image (JPG/PNG) via drag-drop or webcam capture.

🔬
2. Preprocess

OpenCV resizes, normalises, and enhances the image to 224×224 for the CNN.

🧠
3. Predict

MobileNetV2 CNN classifies the image into Fresh, Semi-Fresh, or Rotten.

📊
4. Report

Results include freshness %, shelf life, expiry date, storage tips, PDF & QR.
